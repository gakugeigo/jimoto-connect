## 2. 画面詳細設計

### 2.1 ランディングページ (LP)
- **目的**: サービスの価値を一瞬で伝え、登録を促す。
- **主要要素**:
  - キャッチコピー: 「東京で、地元のあいつと再会しよう。」
  - メインビジュアル: 居酒屋で盛り上がる同郷の人々のイラスト（レトロモダン）。
  - 「Googleで始める」ボタン。

### 2.2 オンボーディング (初回登録)
- **目的**: 検索に必要なデータを確実に入力させる。
- **ステップ**:
  1. **基本情報**: ニックネーム、生まれ年。
  2. **出身地**: 都道府県、市区町村。
  3. **出身校**: 中学・高校の検索と選択。
  4. **部活動**: 選択した学校での部活を入力。
  5. **現在地**: 現在住んでいるエリア。

### 2.3 ホーム (タイムライン)
- **表示内容**:
  - フォローしているユーザーの投稿（将来機能）。
  - 「あなたにおすすめの同郷人」レコメンド。
  - 参加している学校掲示板の最新書き込み。

### 2.4 検索ページ (Core Feature)
- **検索フォーム**:
  - **出身地**: [県選択] [市選択]
  - **現在地**: [県選択] [市選択] [駅名入力]
  - **学校・部活**: [学校名検索] [部活名入力]
  - **年代**: [生まれ年] (±3歳などのオプション)
- **表示モード切り替え**:
  - **リスト表示**: ユーザーカードが一覧で並ぶ。
  - **マップ表示**: 地図上にピンが表示される（詳細位置はぼかす）。

### 2.5 学校掲示板 / 地域掲示板
- **デザイン**:
  - 「居酒屋のノート」や「黒板」をモチーフにした背景やあしらい。
  - 投稿は付箋や手書き風のフォント（読みやすさは維持）で表示。
- **機能**:
  - 新規投稿ボタン（FAB: Floating Action Button）。
  - 「懐かしい！（いいね）」ボタン。
  - コメント機能。

### 2.6 プロフィールページ
- **ヘッダー**:
  - アバター、ニックネーム、出身地バッジ、現在地バッジ。
- **経歴**:
  - 出身校タイムライン（中学 → 高校）。
  - 所属していた部活アイコン。
- **自己紹介**:
  - 自由記述テキスト。

## 3. デザインガイドライン (UI/UX Direction)

### 3.1 カラーパレット
- **メインカラー**: 暖色系のオレンジ・ブラウン（居酒屋の灯り、木目のイメージ）。
  - `#D97706` (Amber 600)
- **アクセントカラー**: レトロな青緑（黒板、学校のイメージ）。
  - `#0F766E` (Teal 700)
- **背景色**: 生成り色、紙の質感。
  - `#FFFBEB` (Amber 50)

### 3.2 タイポグラフィ
- **見出し**: 親しみやすいゴシック体、または少しレトロな明朝体。
- **本文**: 視認性の高いサンセリフ体 (Inter, Noto Sans JP)。

### 3.3 UXのこだわり
- **入力の楽しさ**:
  - 学校名を入力すると、候補がサジェストされる（ストレスフリー）。
  - 登録完了時に「おかえりなさい！」と表示するなど、温かいマイクロコピー。
- **安心感の演出**:
  - 共通点（同じ高校、同じ部活）を強調表示して、「話しかけても大丈夫そう」な空気を作る。
