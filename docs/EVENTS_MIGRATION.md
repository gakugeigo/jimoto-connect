# 同窓会イベント機能のマイグレーション

Phase 4 の同窓会イベント機能を使うには、以下のマイグレーションを実行してください。

---

## マイグレーションファイル

```
supabase/migrations/20260220120000_create_events.sql
```

---

## 実行手順

1. 上記ファイルを開き、SQL をすべてコピー
2. Supabase ダッシュボード → SQL Editor → New query
3. 貼り付けて Run をクリック

（詳細は docs/GROUPS_MIGRATION.md のステップ2〜7 を参照）

---

## 作成されるテーブル

- **events**: イベント情報（タイトル、日時、場所、主催者、関連学校など）
- **event_attendances**: 参加者一覧

---

## 完了後

- ダッシュボードの「同窓会イベント」リンクから `/events` にアクセス可能
- イベント作成・参加申し込み・一覧表示が利用可能になります
