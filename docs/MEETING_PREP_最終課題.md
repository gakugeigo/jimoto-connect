## 7. 技術構成（説明用）

- **アーキテクチャ**: Next.js App Router + Server Actions + Supabase + Clerk
- **認証連携**: Clerk JWT を Supabase に渡し、RLS で `auth.jwt() ->> 'sub'` と照合
- **マイグレーション**: `supabase/migrations/` を SQL Editor で手動実行
