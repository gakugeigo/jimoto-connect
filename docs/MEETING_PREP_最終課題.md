## 7. 技術的な補足（説明用）

- **アーキテクチャ**: Next.js App Router + Server Actions + Supabase + Clerk
- **認証連携**: Clerk の JWT を Supabase に渡し、RLS で `auth.jwt() ->> 'sub'` と照合
- **マイグレーション**: `supabase/migrations/` に複数ファイル。Supabase SQL Editor で手動実行する運用
