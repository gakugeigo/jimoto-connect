# デモデイ発表 準備ガイド

## 開発を止めて発表準備に集中するには

1. **ターミナルで `npm run dev` を止める**
   - Ctrl + C で停止
   - またはターミナルを閉じる

2. **発表資料の作成に使うツールを選ぶ**
   - **Genspark**: この Markdown を入力してスライド生成
   - **Notebook LM**: ドキュメントをアップロードして要約・スライド生成
   - **Cursor**: この Markdown をベースに HTML スライド（reveal.js 等）を生成

---

## 発表資料の元データ

**`docs/DEMO_DAY_PRESENTATION.md`** にスライド用の内容を用意しました。

- プロダクト概要（課題・解決策・主要画面）
- 技術構成（アーキテクチャ、使用サービス）
- ビルド&デプロイで工夫した点
- 今後のロードマップ
- デモの流れ

このファイルを **Genspark** や **Notebook LM** に読み込ませて、スライド形式に変換してください。

---

## ツール別の使い方

### Genspark
1. https://genspark.ai/ にアクセス
2. 「プレゼンテーション作成」や「スライド生成」を選択
3. `DEMO_DAY_PRESENTATION.md` の内容をコピペして投入
4. 生成されたスライドをダウンロード・編集

### Notebook LM
1. https://notebooklm.google.com/ にアクセス
2. ノートブックを作成
3. `DEMO_DAY_PRESENTATION.md` をアップロード
4. 「プレゼンテーションを生成」などの機能でスライド作成

### Cursor で HTML スライド
- 「DEMO_DAY_PRESENTATION.md を reveal.js 形式の HTML スライドに変換して」と依頼
- 生成された HTML をブラウザで開いて発表

---

## デモの準備

### 実機操作の場合
- 本番 URL を開いておく
- テストアカウントでログイン済みにしておく
- 画面共有 or プロジェクター接続を確認

### 録画の場合
- OBS / QuickTime / Windows の「ゲームバー」などで画面録画
- 1〜2分程度の短いデモ動画に編集
- スライドに埋め込む or 別タブで再生
