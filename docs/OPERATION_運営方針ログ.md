## 2025年2月21日: 悪意ユーザー対応の方針

### 決定内容

**複数からの報告があった場合、その人を削除することを検討する**

### 詳細

- 1件の報告だけでは誤認・嫌がらせの可能性があるため、複数報告を閾値とする
- 「複数」= 同じユーザーに対する報告が、**異なる報告者から2件以上**
- 「検討」= 報告内容と該当ユーザーの状況を確認し、警告・所属削除・アカウント削除のいずれかを判断する
- 同一人物による複数回の報告は1件としてカウント（水増し防止）

### 対応の流れ（案）

1. 複数（2件以上）の報告を受ける
2. 報告内容・該当ユーザーの投稿・プロフィールを確認
3. 運営が判断（警告 / 該当学校の所属削除 / アカウント削除）
4. 必要に応じて本人に連絡・説明の機会を設ける
