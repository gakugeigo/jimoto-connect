# ビジネス要件：Jimoto-Connect（仮）

## 1. 課題定義
### 解決する課題
- **都会での孤独感・疎外感**: 地方から進学や就職で上京してきた人が、物理的な「知り合い」はいても、心理的に安心できる「同郷の仲間」が近くに見つからない。
- **地元コミュニティの希薄化**: 地元の友人がどこに住んでいるかわからず、疎遠になってしまう。
- **既存SNSの限界**: FacebookやInstagramでは「今、近くに住んでいる同郷の人」という条件（過去×現在）での検索が難しく、偶然の再会に頼るしかない。

### 課題の深刻度
- **中〜高**: 深刻なメンタルヘルス不調とまではいかなくとも、日常的な「寂しさ」や「話し相手がいない」ストレスは多くの移住者が抱えている。特に緊急時（災害時など）や、ふと方言で話したい時に強く感じる。

### 課題が発生する状況
- 上京直後の心細い時期。
- 職場と家の往復だけの生活に疲れた時。
- 地元のニュースや方言に触れて懐かしくなった時。

## 2. 市場環境
### 既存の代替手段
- **同窓会サイト/アプリ**: 過去のつながり重視で、「今の居住地」でのマッチングに弱い。
- **Facebook/Mixi**: コミュニティ機能はあるが、アクティブ率が下がっていたり、実名制のハードルが高かったりする。
- **新橋の「同窓会居酒屋」のようなリアル店舗**: 物理的にそこに行かないといけない。数が限られる。

### 差別化ポイント（競争優位性）
1.  **「出身地」×「現在地」のクロス検索**:
    - 「沖縄県・恩納村出身」かつ「東京都・目黒区在住」のような、ピンポイントな共通点を持つ人を探せる。
2.  **必須入力による「濃い」コンテキスト**:
    - 出身中学・高校の登録を必須（または強く推奨）にすることで、曖昧な「出身」ではなく、共通の話題（先生、行事、校歌）を持つ相手と確実につながれる。
    - **部活動データの活用**: 「同じ高校の野球部」だけでなく、「沖縄県の高校で野球部だった同い年」のような、学校を跨いだ共通点でもつながれるようにする（例：夏の大会で対戦したかも？という盛り上がり）。
3.  **デジタル寄せ書き（居酒屋ノート体験）**:
    - 母校のページに「2010年卒の山田です、今中目黒で飲んでます！」のように気軽に書き込めるフロー型の掲示板機能。

## 3. ターゲットユーザー
### メインペルソナ
- **属性**: 地方出身で、現在は東京（または大阪・福岡などの大都市）に住む20代〜30代の社会人。
- **状況**: 仕事には慣れてきたが、休日に遊ぶ友人が会社の同期くらいしかいない。
- **ニーズ**: 「気を使わずに方言で話したい」「地元のローカルネタで盛り上がりたい」。

### サブペルソナ
- **Uターン・Iターン検討者**: 都会出身だが、地方に移住してその土地の人と繋がりたい人。

## 4. ビジネスモデル
### 収益モデル（案）
1.  **地域広告（将来構想）**:
    - ユーザーの「出身地」に関連する特産品やイベントの広告（例：「沖縄物産展in新宿」の広告を、沖縄出身・東京在住者に出す）。
2.  **プレミアム機能**:
    - 足跡帳への書き込み放題、検索条件の保存、非公開モードなど。

## 5. リスクと仮説
### 検証すべき仮説
- 「出身校や出身地をネット上で公開すること」への抵抗感が、安心感（同郷という絆）を上回るか？
    - → **対策**: 公開範囲の制御（同郷の人にしか見せない等）や、認証システムの導入を検討。
- 実際に「会いたい」と思うか、それとも「ネット上で懐かしむ」だけで満足するか？
    - → **対策**: オフライン交流を促す「県人会イベント」機能などで後押し。

## 6. プロダクト要件（概要）
### コア機能（MVPスコープ）
1.  **プロフィール登録**:
    - 出身地（都道府県・市区町村）、出身校（中・高）、**所属部活**（野球、吹奏楽など）、現在の居住地（最寄り駅・市区町村）、生まれ年。
2.  **検索機能**:
    - 地図またはリストで「近くの同郷人」を探す。
    - **クロス検索**: 「エリア×部活×年代」などで絞り込み（例：東京在住 × 沖縄出身 × 高校野球部 × 1995年生まれ）。
3.  **母校/地元掲示板（デジタル居酒屋ノート）**:
    - 学校や地域ごとのスレッド機能。
4.  **認証/ログイン**:
    - Clerkを使用したソーシャルログイン（Google等）。

### UI/UXの方向性
- **レトロ×モダン**: 「懐かしさ」を感じさせる温かみのあるデザイン（茶色や紙の質感など）だが、操作性はモダンでサクサク動く。
